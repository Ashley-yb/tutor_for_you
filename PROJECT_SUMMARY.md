#  给你讲题的同桌 - 项目说明

## 🎯 项目简介

"给你讲题的同桌"是一个基于 AI 的高中题目智能解答助手，帮助学生快速获取题目解析。

### 核心功能

-  **图片识别**：自动识别题目图片中的文字和公式
- 🤖 **智能解答**：AI 教师生成详细的解题步骤
- 🎯 **多学科支持**：数学、语文、英语、物理、化学等
-  **两种模式**：
  -  快速模式：一步到位，直接识别并解答（推荐）
  - 📝 两步模式：先查看识别结果，再生成解答

---

## 🏗️ 技术架构

### 前端
- **框架**：Streamlit
- **特点**：纯 Python 开发，无需前端知识
- **部署**：Streamlit Cloud（免费）

### 后端服务
1. **通义千问 VL**（主要）
   - 模型：qwen-vl-plus
   - 功能：视觉语言模型，直接识别图片并解答
   - API：阿里云 DashScope
   
2. **MinerU**（可选）
   - 功能：专业 OCR 识别
   - API：OpenXLab
   - 用途：两步模式中的识别环节

3. **通义千问 Turbo**（可选）
   - 模型：qwen-turbo
   - 功能：文本大模型
   - 用途：两步模式中的解答环节

### 工作流程

```
用户上传图片
    ↓
选择处理模式
    ↓
┌─────────────┬─────────────┐
│  快速模式    │  两步模式    │
│             │             │
│ 通义千问VL  │ MinerU识别  │
│ 识别+解答   │     ↓       │
│             │ 通义千问解答│
└─────────────┴─────────────┘
    ↓
显示详细解答
    ↓
可下载保存
```

## 🔑 API 配置

### 用户需提供

应用采用 **BYOK（Bring Your Own Key）** 模式，用户需要自己提供 API Key：

1. **通义千问 API Key**（必须）
   - 获取地址：https://dashscope.aliyun.com/
   - 费用：约 0.01-0.03 元/次
   - 新用户有免费额度

2. **MinerU API Token**（可选）
   - 获取地址：https://openxlab.org.cn/
   - 费用：约 0.1-0.5 元/次
   - 仅两步模式需要

### 安全机制

- ✅ API Key 仅在会话期间使用
- ✅ 关闭页面后自动清除
- ✅ HTTPS 加密传输
- ✅ 不保存到服务器

##  部署方式

### 方案一：Streamlit Cloud（推荐）

**优点：**
- 完全免费
- 5分钟搞定
- 自动 HTTPS
- 无需维护服务器

**步骤：**
1. 上传代码到 GitHub
2. 在 Streamlit Cloud 创建应用
3. 分享链接给用户

详见：[QUICK_DEPLOY.md](QUICK_DEPLOY.md)

### 方案二：Vercel

**优点：**
- 免费额度充足
- 支持自定义域名
- 全球 CDN 加速

**适用场景：**
- 需要自定义域名
- 对性能要求较高

### 方案三：云服务器

**优点：**
- 完全控制
- 可定制性强

**缺点：**
- 需要付费
- 需要运维

**适用场景：**
- 大规模商用
- 需要深度定制
